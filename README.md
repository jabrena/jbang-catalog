# jbang-catalog

A collection of JBang scripts.

- [jbang-catalog.json](jbang-catalog.json)

## How to use it?

**Prerrequisite:** Jbang installed

```bash
jbang cache clear
jbang catalog list
jbang catalog list jabrena

jbang setup@jabrena
jbang colossal-cave-adventure@jabrena
jbang raton-loco@jabrena
```
